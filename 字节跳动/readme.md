  渲染树 DOM + CSS树 表示DOM节点如何显示

  当页面上的DOM节点发生非布局变更时GPU需要再次绘制一次  叫做重绘

  - 页面首次渲染
  - 添加或删除可见的DOM元素
  - 改变元素位置
  - 改变元素尺寸
  - 改变窗口大小
  - offset...  client...
